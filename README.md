# Simple Cart Fees

Add extra fees to your WooCommerce checkout. Simple, flexible, and works with both classic and block checkout.

## Features

- Create unlimited fees
- Required or optional (checkbox for customers)
- Conditional fees (minimum cart amount)
- Full tax support
- Block checkout compatible
- HPOS compatible

## Requirements

- WordPress 6.0+
- WooCommerce 8.0+
- PHP 7.4+

## Installation

1. Download and upload to `/wp-content/plugins/`
2. Activate the plugin
